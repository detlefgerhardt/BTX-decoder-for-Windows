# BTX-decoder-for-Windows
BTX decoder for Windows 10/11 in C#/.NET8 with SDL3 UI

[GitHub Pages](https://pages.github.com/)
This project is based on he [btx_decoder project]((https://github.com/bildschirmtext/btx_decoder)/) by Michael Steil.

It was generated by AI (OpenAI Codex 5.0) from the original C project. 
So it's now a C#/.NET8 application with SDL3 UI for Windows 10/11.
Caution - 100% Vibe Coding. No manual checks or modifications were made.

The UI is very minimalistic.

Installation and start:
- Extract the ZIP file from binaries into a directory.
- Install .NET8 if not already installed.
- Get the latest SDL3-Version from https://github.com/libsdl-org/SDL/releases,
  extract SDL3.dll and put it into the native\win-x64 directory.
- Start with **BtxDecoder server-address:port**
  (e.g. **BtxDecoder 195.201.94.166:20000**)

F1 = '*' / F2 = '#'
